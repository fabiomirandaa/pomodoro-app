# PomodoroApp

> Aplicação de pomodoro para organização pessoal na realização de tarefas. Consiste de um Timer para contagem dos Pomodoros e uma seção de Todos para registrar as tarefas que precisam ser realizadas.

# Tecnologias em uso

- [Angular](https://angular.io/)
- [RxJS](https://rxjs-dev.firebaseapp.com/)
- [Typescript](https://www.typescriptlang.org/)
- [NGRX](http://ngrx.io/)
- [Bulma](https://bulma.io/)
